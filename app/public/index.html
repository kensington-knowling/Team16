Welcome to the Indiana Soccer Referees Association Homepage!


What is the Indiana Soccer Referees Association?


The Indiana Soccer Referees Association (ISRA) assigns referees to club matches in the greater-Indianapolis area. Correctly assigning multiple referees across games is vital and done properly through this online tool.


Your Contact:


If you need to contact someone about the efficiency of this tool, Theas Signor can help! Ms. Signor can help with any questions that arise.


This Application:


This application is a proof of concept for a future secure application for ISRA. If you are interested in using this application as an open source tool, contact Ms. Signor for project updates and availability.


This application will help you with the following tools:

  • Upcoming Games
  • Referees
  • Game Assignments
  • Game & Referee Reports
